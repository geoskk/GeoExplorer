GeoExplorer
===========

A Reverse-Geocoding tool made with HTML5.

To touch the published page, go to the url below.  
http://poppycocker.github.io/GeoExplorer/

Viewable Maps:
* Google Maps
* OpenStreetMap

Powered by
* Google Maps API v3
* Leaflet.js
* jQuery
* Backbone.js
* Underscore.js
* Typicons
* shortcut.js
